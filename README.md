CHAT – Full-Stack Chatbot Application

CHAT is a full-stack chat application inspired by modern conversational AI systems.  
It includes a **frontend interface** and a **backend server** to handle messages and responses.

---

 Key Features

-  Clean and interactive chat UI  
-   Real-time message sending and receiving  
- Frontend + backend integration  
- Simple setup and lightweight design
